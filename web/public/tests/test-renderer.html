Terminal Renderer Test

1. Test with Pre-recorded Cast File

2. Test with Live Session Stream

3. Manual Test

Usage Instructions:

  • Cast File Test: The default URL points to your test cast file. Click "Load Cast File" to render it.
  • Stream Test: Enter a session ID and click "Connect to Stream" to watch live terminal output.
  • Manual Test: Enter raw ANSI escape sequences to test specific rendering features.

Features Supported:

  • ✅ ANSI color codes (standard 16 colors + 256-color + RGB)
  • ✅ Text formatting (bold, italic, underline, strikethrough)
  • ✅ Cursor movement and positioning
  • ✅ Screen clearing and line erasing
  • ✅ Alternate screen buffer support
  • ✅ Scrollback buffer (up to 1000 lines)
  • ✅ Scroll regions
  • ✅ Auto-wrap mode
  • ✅ Inverse video
  • ✅ Cast file format v2 parsing
  • ✅ Server-Sent Events streaming
